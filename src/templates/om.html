Kitchen Guard System
Who are we?

In many households with elderly residents, kitchen-related accidents—such as unattended cooking or forgotten appliances—pose significant safety risks. Age-related cognitive decline, memory lapses, and reduced mobility can make it difficult for individuals to respond quickly to such hazards. As populations age and more people express a strong desire to remain in their own homes for as long as possible, there is a growing need for intelligent, non-intrusive safety systems. These systems should offer passive monitoring and timely alerts to help prevent and handle accidents, enabling elderly individuals to live independently while remaining as safe as possible.

This project addresses this need by developing a smart kitchen guard system designed specifically for elderly users. The system is capable of detecting potentially dangerous situations and notifying caregivers or relatives in real time.

To achieve this, the system integrates Zigbee-enabled sensors to monitor kitchen activity and presence, a Raspberry Pi for local processing, and a Rust-based web-server for managing alerts and remote notifications. Communication between components is handled using the MQTT protocol.
